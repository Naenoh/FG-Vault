# fgctech

### Objectives

- Make it easy to archive and find tech


### Requirements

- Multi games
- Easy to use
- No auth
- No duplicates
- Fast

### Stack

- Flask/SQLAlchemy/Graphene(GraphQL)/Postgres
- Vue/Apollo(GraphQL)/Bulma
